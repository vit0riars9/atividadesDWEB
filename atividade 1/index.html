Página do Curso de Desenvolvimento Web

O curso de Desenvolvimento Web tem como objetivo ensinar tecnologias modernas para criação de sites e sistemas web.

Conteúdo do curso

  • HTML básico e semântico
  • CSS e layout responsivo
  • Introdução ao JavaScript
  • Versionamento com Git

Informações importantes

  • Duração: 3 meses
  • Carga horária: 120 horas
  • Certificado incluso

Inscrição

Inscrições podem ser realizadas por meio de:

  1. Formulário enviado por email
  2. Comparecimento presencial à secretaria
